JS精粹

2020-02-09

基本数据类型

  • Undefined Null Boolean String Number
  • 而Object则是复杂数据类型

typeof 返回的数据类型在基本数据类型的基础(undefined object boolean number string symbol注意没有null)上还有function

注释

  • 避免使用/**/ 这种区块注释，容易与正则表达式混杂后失去效果

标识符

  • 字母开头，然后用字符、数字、下划线_拼凑
  • 被用于语句、变量、参数、属性名、运算符和标记(位置)

数字

  • 100 === 1e2
  • 内部无论整型还是浮点数都表示为64为的浮点数1 === 1.0
  • typeof NaN === 'number'
  • Infinity表示所有大于1.79769313486231570e+308的值
  • Math.floor(213.213) 结果为 213转化整型

字符串

  • 用\u 约定指定数字字符编码，比如"A" === "\u0041"

运算优先级

  • 三元运算符是排在最后才运算的，所以如果需要讲三元运算符结果与其他字符串拼凑应该用括号括起来

Object

  • 对象通过引用来传递，永远不会被复制

    基本类型是存放在栈内存中的简单数据
    引用类型(object array)是存放在堆内存中的对象，被变量申明的时候是将一个引用放入栈内存中

  • 对象的修改不涉及其原型，修改继承对象的属性会覆盖原型的属性，属性的寻找是从对象开始然后往原型链上去找

  • 反射-过滤通过原型链索引的属性,枚举for in
    一种通过typeof flgight.number=== 'funtion' 的方式判断属性是否是需要的数据
    一种通过flight.hasOwnProperty('number') 的方式判断是否是对象独有属性

  • 删除
    不会涉及到原型属性

  • 减少全局变量污染
    将所有全局变量都放入一个命名空间下，声明唯一的全局变量来存放
    还可以通过闭包来进行信息隐藏，减少全局变量污染

函数

  • 函数在创建时传递隐藏属性-函数的上下文和实现函数行为的代码

  • 每个函数在创建时候也配有一个prototype属性，其值为拥有constructor属性且其值即为该函数的对象，函数与其他object的不同之处在于可以被调用

  • 函数可以访问自己的参数变量，同时也能自由访问把它嵌套在其中的父函数的参数与变量，通过函数字面量创建的函数对象包含一个连接到外部上下文的连接，这就是闭包

  • 除了声明时候定义的形式参数，函数还接受两个附加的参数this arguments，this取决于函数的调用方式

    • 调用模式

      • 方法调用模式：函数被保存为一个对象的属性时被称为方法。方法被调用(obj.fun | obj[‘fun’])的时候this绑定到该对象。通过this可取得他们所属对象上下文的方法被成为公共方法

      • 函数调用模式：当一个函数并非一个对象的属性时，他就是被当做一个函数来调用的。使用函数调用的时候this对象被绑定到全局对象而不是外部的this变量(js的设计错误)，所以一个函数内部定义的函数调用时候，可以通过定义一个that变量赋值this，这样内部函数就可以通过这个变量访问外部this了

      • 构造器调用模式：通过new的方式去继承一个函数，这样的方式会创建一个连接到该函数的prototype成员的新对象，this会绑定到该新对象上，注意的是被new的函数本身的代码会作为构造器函数，可用来构造属性

        但是本方法不是书中推荐的，在之后有更好的替代方法

      • Apply调用模式：通过apply方法
  • 调用运算符-([,arguments]),实际参数arguments个数比形式参数paramenters过多时多余参数会被忽略，过少时缺失值会被替换成undefined，参数值不会进行类型检查
