app.enable(‘view cache’);
});

socket.io

消息传递基于传输并非完全依靠websocket
没有数据返回的时连接还会持续打开20-50秒
解决了websocket被代理或防火墙禁止的问题
默认断开重连

此外允许单个连接中利用命名空间来将消息彼此分开来
DEBUG=socket.io* node myapp可以看到调试信息
